## About

Tired of browsers pushing ads, irrelevant news, pop-ups, and banners?  
I was too. So I built EsthevanSearch.  

A simple startpage that shows only what matters:  
- Time  
- Weather  
- Search  
- Useful shortcuts  

Fast, lightweight, and distraction-free.
